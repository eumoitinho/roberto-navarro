## Por que existe

Roberto Navarro é referência em coaching financeiro no Brasil, com mais de **300.000 vidas transformadas** através de seus métodos exclusivos de mentalidade.
A plataforma foi construída para dar **autonomia total de conteúdo** à equipe via Payload CMS, com e-commerce integrado para monetização direta.

---

## O que a plataforma entrega

- **Página institucional**: apresentação do ICF, método, depoimentos e resultados
- **E-commerce integrado**: venda de livros, cursos online e programas de mentoria
- **Checkout seguro**: fluxo de compra completo dentro do site
- **CMS headless**: Payload CMS para gestão autônoma de todo o conteúdo
- **Responsivo**: otimizado para conversão em mobile e desktop

---

## Stack

- **Frontend**: Next.js 14 (App Router), React, Tailwind CSS
- **CMS**: Payload CMS (headless, self-hosted)
- **E-commerce**: checkout integrado com gestão de produtos
- **Deploy**: Vercel
